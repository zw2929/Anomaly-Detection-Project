Section 3: Using Autoencoder ¶

An autoencoder is a type of neural network that learns to compress data into a smaller representation and then reconstruct it back to its orginal form. It consists of an encoder that reduces the data's dimensionality and a decoder that reconstructs the data from this compressed representation. Also, it captures outliers by learning to reconstruct typical patterns in the data with low reconstruction error, while anomalies have a high reconstruction error, which are much differ from the normal patterns.¶

Section 3.1: Using Autoencoder for Feature1-9 ¶

Here, I divided my features into two groups: features 1-9 and features 10-15, because I noticed that these two groups have different scopes. Specifically, features 1-9 focus on comparing hospital payment and discharge ratios relative to state or city benchmarks, while features 10-15 concentrate on similar ratios but specifically for Medicare payments and payments per discharge. This distinction allows for a more targeted analysis of general payment patterns versus Medicare-specific and discharge-specific payment behaviors. For our anomalies detection here, we mainly foucs on features 1-9.¶

def descriptive_stat_threshold(X, scores, threshold):
    normal_mask = scores <= threshold
    anomaly_mask = scores > threshold

    normal_stats = X[normal_mask].describe()
    anomaly_stats = X[anomaly_mask].describe()

    summary_df = pd.DataFrame({
        'Group': ['Normal', 'Outlier'],
        'Count': [normal_mask.sum(), anomaly_mask.sum()],
        'Count %': [normal_mask.mean() * 100, anomaly_mask.mean() * 100]
    })
    feature_name=['Hospital_Average_Total_Payments','City_to_State_Payment_Ratio','Provider_to_State_Payment_Ratio', 'Hospital_to_State_Discharge_Ratio','City_to_State_Discharge_Ratio','Provider_to_State_Discharge_Ratio', 'Hospital_to_State_Medicare_Payment_Ratio','City_to_State_Medicare_Payment_Ratio','Provider_to_State_Medicare_Payment_Ratio']
    for i, feature in enumerate(feature_list, 1):
        summary_df[feature_name[i-1]] = [
            X[normal_mask][feature].mean(),
            X[anomaly_mask][feature].mean()
        ]


    summary_df['Anomaly_Score'] = [
        scores[normal_mask].mean(),
        scores[anomaly_mask].mean()
    ]
    summary_df=summary_df.T
    summary_df.columns = summary_df.iloc[0]
    summary_df=summary_df.iloc[1:]
    return summary_df

Aggregating to achieve model stability¶

Using multiple autoencoder models to improve the stability of outlier detection by aggregating results. This aggregation reduces the sensitivity to variations in individual models and improves the overall reliability of detecting abnormal hospital billing patterns.¶

Section 4: Using Isolation Forest ¶

Isolation Forest is an algorithm that detects outliers by randomly cuts data into small pieces to see which points are stand out from the rest. It works by randomly splitting the data multiple times. Points that are easier to separate from others like outliers end up isolated faster. The algorithn assigns an anomaly score to each point, and those with higher scores are identifies as potential outliers.¶

Achieve Model Stability by Aggregating Multiple Models¶

Using mutiple isolation forest models with different numbers of estimators k_list to improve the stability of outlier detection. Each model is trained on the data, and outlier scores are calculates for both training and testing sets. The scores are then standardized to allow consistent comparison, aiming to achieve more reliable anomaly detection by combining results from multiple models.¶

from pyod.utils.utility import standardizer
from pyod.models.iforest import IForest

k_list = [100, 200, 300, 400, 500]
n_clf = len(k_list)

train_scores = np.zeros([X_train.shape[0], n_clf])
test_scores = np.zeros([X_test.shape[0], n_clf])

# Modeling
for i in range(n_clf):
    k = k_list[i]
    isft = IForest(contamination=0.05, n_estimators=k)
    isft.fit(X_train)

    train_scores[:, i] = isft.decision_function(X_train)
    test_scores[:, i] = isft.decision_function(X_test)
    
train_scores_norm, test_scores_norm = standardizer(train_scores,test_scores)
